📄 صياغة ملف JSON للاستيراد

عند إضافة محتوى بالدفعة، يجب أن يكون الملف بالتنسيق التالي: 

[
  {
    "title": "عنوان الموضوع",
    "content": "نص الموضوع هنا..."
  }
]

تأكد من أن كل عنصر يحتوي على المفتاحين title و content فقط لضمان التوافق.
