# تطبيق الذاكرين (Al-Dhakereen)

تطبيق إسلامي احترافي مبني باستخدام Flutter، يتميز بتصميم أنيق وفخم، ويدعم العديد من الميزات الإسلامية.

## الميزات الرئيسية
- **أوقات الصلاة:** حساب دقيق لأوقات الصلاة بناءً على الموقع الجغرافي (مع دعم خاص للمحافظات العراقية).
- **القرآن الكريم:** عرض السور بتنسيق فاخر مع خطوط قرآنية احترافية.
- **الأدعية والزيارات:** قاعدة بيانات شاملة مصنفة حسب الأيام والمناسبات.
- **المسبحة الإلكترونية:** عداد أذكار تفاعلي مع اهتزاز وحفظ تلقائي.
- **تخصيص كامل:** تغيير سمة التطبيق، الألوان، الخطوط، الخلفيات، والوضع الليلي/النهاري.
- **تنبيهات الأذان:** دعم كامل للأذان والتنبيهات لكل صلاة.

## كيفية إضافة أدعية أو محتوى جديد

يتم إدارة محتوى التطبيق من خلال ملف JSON مركزي موجود في المسار التالي:
`assets/data/content.json`

### هيكلية الملف:
الملف يحتوي على قسمين رئيسيين:
1. `sections`: لتعريف الأقسام (مثل القرآن، الأدعية، الزيارات).
2. `duas_days`, `duas_taqeebat`, `duas_general`, إلخ: قوائم المحتوى لكل قسم.

### خطوات الإضافة:
1. افتح ملف `assets/data/content.json`.
2. لإضافة دعاء جديد في قسم "الأدعية العامة":
   - ابحث عن مفتاح `"duas_general"`.
   - أضف كائنًا جديدًا في نهاية القائمة بالشكل التالي:
     ```json
     {
       "id": 100,
       "title": "عنوان الدعاء الجديد",
       "content": "نص الدعاء بالكامل هنا..."
     }
     ```
3. لإضافة أعمال يوم معين (ليتم عرضها تلقائيًا في الصفحة الرئيسية):
   - ابحث عن مفتاح `"duas_days"`.
   - تأكد أن العنوان يحتوي على اسم اليوم (مثلاً: "دعاء يوم السبت").
   - سيقوم التطبيق تلقائيًا بالتعرف على اليوم وعرضه في بطاقة "دعاء اليوم" في الصفحة الرئيسية.

### ملاحظات هامة:
- تأكد من صحة تنسيق JSON (استخدام الفواصل بشكل صحيح).
- استخدم علامة `\n` للسطر الجديد داخل نص المحتوى.
- بعد تعديل الملف، سيتم تحديث المحتوى تلقائيًا عند تشغيل التطبيق (أو عبر ميزة المزامنة السحابية إذا كانت مفعلة).

## متطلبات التشغيل
- Flutter SDK >= 3.4.0
- نظام تشغيل أندرويد (دعم 64 بت فقط لضمان الأداء العالي).

## تطوير
تم تطوير هذا التطبيق ليكون سلاحًا إيمانيًا في يد كل مسلم، مع التركيز على سهولة الاستخدام والأناقة.
