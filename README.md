# تطبيق الذاكرين (AlDhakereen)

تطبيق إسلامي احترافي مبني باستخدام تقنيات الويب الحديثة ومحول إلى تطبيق Android أصيل (64-بت) باستخدام Capacitor.

## المميزات
- تصميم فخم وأنيق (ذهبي وأسود).
- دعم كامل للوضع الليلي والنهاري.
- التحكم في حجم الخط لسهولة القراءة.
- إمكانية إضافة أدعية وزيارات وملاحظات جديدة.
- ميزة البحث السريع.
- ميزة النسخ والمشاركة.

## طريقة إضافة محتوى جديد (أدعية أو زيارات)

لإضافة محتوى جديد للتطبيق، اتبع الخطوات التالية:

### 1. من خلال واجهة التطبيق (للمستخدم):
- افتح القسم المطلوب (الأدعية، الزيارات، أو الملاحظات).
- اضغط على زر الإضافة (+) الموجود في الشريط العلوي جهة اليسار.
- سيظهر نموذج الإضافة، قم بإدخال "العنوان" و "المحتوى".
- اضغط على زر "حفظ". سيتم تخزين البيانات محلياً في جهازك وستظهر في القائمة فوراً.

### 2. من خلال الكود المصدري (للمطور):
إذا كنت ترغب في إضافة محتوى افتراضي يظهر لجميع المستخدمين عند تثبيت التطبيق:
- افتح الملف `www/js/app.js`.
- ابحث عن متغير `data`.
- قم بإضافة كائن جديد داخل المصفوفة المناسبة (`duas` أو `visits`).
- مثال:
```javascript
duas: [
    {
        id: Date.now(),
        title: 'عنوان الدعاء الجديد',
        content: 'نص الدعاء هنا...'
    },
    ...
]
```

## بناء التطبيق
تم ضبط التطبيق ليدعم معمارية 64-بت فقط (`arm64-v8a`, `x86_64`) لضمان أقصى أداء وتوافق مع متطلبات متجر Google Play الحديثة.

لمزيد من المعلومات، يرجى مراجعة الكود المصدري.
