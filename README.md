# تطبيق الذاكرين (Al-Dhakereen)

تطبيق إسلامي احترافي مبني باستخدام Flutter، يتميز بتصميم فاخر وأداء سلس، مع دعم كامل للغة العربية والوضع الليلي.

## المميزات الاحترافية
- **تصميم فاخر:** مزيج من الأسود والذهبي مع زخارف إسلامية هندسية (نجمات ثمانية).
- **موسوعة شاملة:** القرآن الكريم، الصحيفة السجادية، الأدعية، الزيارات، والأذكار.
- **تخصيص كامل:** التحكم في ألوان التطبيق، شفافية الواجهة، وحجم الخط.
- **أوقات الصلاة:** حساب دقيق لأوقات الصلاة في العراق (مع التركيز على الحلة وبغداد) ودعم التعديل اليدوي.
- **تحديث تلقائي:** يتم تحديث المحتوى تلقائياً من السحابة عند توفر إصدارات جديدة.

## الهندسة التقنية
- **64-بت فقط:** التطبيق مصمم ليعمل بأعلى كفاءة على معالجات 64-بت (`arm64-v8a`, `x86_64`).
- **Material 3:** استخدام أحدث معايير التصميم من Google.
- **مزامنة صامتة:** جلب البيانات في الخلفية دون إزعاج المستخدم.

## كيفية إضافة أدعية أو محتوى جديد

يمكنك إضافة محتوى جديد للتطبيق عبر طريقتين:

### 1. عبر ملف `content.json` (للمطورين)
قم بتعديل ملف `assets/data/content.json` وإضافة المدخلات الجديدة في قسم `content`:
```json
"duas": [
  {
    "id": 100,
    "title": "عنوان الدعاء الجديد",
    "content": "نص الدعاء هنا..."
  }
]
```

### 2. عبر لوحة التحكم السحابية (CMS)
إذا كنت تمتلك صلاحية الوصول إلى المستودع السحابي المرتبط بالتطبيق:
1. قم بتعديل ملف `content.json` على GitHub.
2. سيقوم التطبيق تلقائياً بجلب التحديثات عند فتحه من قبل المستخدمين عبر محرك المزامنة المدمج.

## ملاحظات البناء (Build)
لإنشاء نسخة APK احترافية:
```bash
flutter build apk --release --target-platform android-arm64,android-x64
```
هذا يضمن أن التطبيق يعمل بنظام 64-بت فقط كما هو مطلوب.
